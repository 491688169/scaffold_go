├── configs 配置文件
├── docs 文档集合
├── global 全局变量
├── go.mod
├── internal 内部模块
│ ├── dao 数据访问层，所有与数据相关的操作，如 MYSQL，ES 等
│ ├── middleware HTTP 中间件
│ ├── model 模型层，用于存放 model 对象
│ ├── routers 路由相关逻辑
│ └── service 项目核心业务逻辑
├── pkg 项目相关的模块包
├── scripts 各类安装、构建、分析等操作的脚本
├── storage 项目生成的临时文件
└── third_party 第三方的资源工具，如 Swagger_UI
